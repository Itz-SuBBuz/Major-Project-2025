Project Overview

The Smart Fan Control System is an IoT-based system designed to control a DC fan through a web interface on any smart device like mobile, laptop, tablet etc. This project enhances comfort and energy efficiency. while allowing remote operation via smartphone connectivity.

Objective

The goal of this project is to design and implement a fan control system that can smoothly vary the fan speed using Pulse Width Modulation (PWM) and be controlled wirelessly through a mobile app interface.

Circuit Design & Working Principle

The ESP32 generates PWM signals that are sent to the BTS7960 motor driver to control motor speed. The mobile app sends commands via Wi-Fi or Bluetooth to the ESP32, which adjusts the duty cycle of PWM. The Hall Effect sensor provides feedback for speed monitoring, ensuring stability and protection from overcurrent.

Software & Programming

The code is written in Arduino IDE using the ESP32 library and Blynk framework for communication. The main logic includes reading sensor data, controlling PWM signals, and updating fan speed based on app commands. 

Setup:
 - Initialize Wi-Fi and Blynk connection
 - Configure PWM output pins
Loop:
 - Read input from app
 - Map values to PWM duty cycle
 - Send feedback to app
